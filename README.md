Hatchet Mod
Replaces the pickaxe with a Hatchet — a single tool that handles both wood-cutting and rock-breaking. Inspired by Story of Seasons: The Grand Bazaar.
What it does
Removes the pickaxe from your inventory and replaces it with a Hatchet
The Hatchet chops trees and breaks rocks, so you never need to swap tools
The axe remains unchanged and unaffected
The Hatchet has its own sprite: a wood-cutting pose when swinging at trees/logs, a rock-breaking pose when swinging at stone
Starting a new game
Both the starting axe and starting pickaxe are removed. You receive a Basic Hatchet instead. The axe is gone from the start — the Hatchet handles all clearing.
Installing on an existing save
Your pickaxe is automatically replaced by a Hatchet at the same upgrade level. Your axe is untouched.
Upgrading
Bring your Hatchet to Clint. Upgrades follow the same progression as a pickaxe (copper → steel → gold → iridium) but cost 8 bars and 150% of the standard gold price:
Upgrade	Bars	Gold
Copper Hatchet	8 copper	3,000g
Steel Hatchet	8 iron	7,500g
Gold Hatchet	8 gold	15,000g
Iridium Hatchet	8 iridium	37,500g
Disabling the mod
Toggle the mod off via the config file or Generic Mod Config Menu. Your Hatchet is replaced by a Pickaxe at the same upgrade level. If you started this save with the mod active, a Basic Axe is also restored. Function switches immediately; the item and sprite revert on next game restart.
Requirements
SMAPI 4.0+
Stardew Valley 1.6+
Generic Mod Config Menu (optional — for in-game toggle)
Compatibility
Single-player only (multiplayer farmhand support not yet implemented)
No Content Patcher required
